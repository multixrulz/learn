Activity C3 — Digital outputs

Level 1 — Use a digital output

Complete the challenge

Your entire setup() function should look like this:

void setup() {
    pinMode(5, OUTPUT);    // set up pin 5 as an output.
}

Your entire loop() function should look like this:

void loop() {
    digitalWrite(5, HIGH);  // turn the LED on
    delay(100);             // wait for 1 tenth second
    digitalWrite(5, LOW);   // turn the LED off
    delay(1000);            // wait for 1 second
}
