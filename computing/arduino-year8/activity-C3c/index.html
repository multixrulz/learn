Level 1 — Change the brightness of an LED

What are we doing in this level, sir?

You are going to learn how to use the analogue output to change the brightness of an LED.

Open a new Arduino window with a blank sketch in it.

Copy the sketch below into your Arduino window. Remember to delete the blank sketch first.

int ledPin = A9;                // define the pin for LED D9 on the ThinkerShield
int buttonPin = 7;              // define the pin for button D7
int buttonState = 0;            // create a variable for what the button is doing

void setup() {
    pinMode(ledPin, OUTPUT);    // set up the ledPin as an OUTPUT
    pinMode(buttonPin, INPUT);  // set up the buttonPin as an INPUT
}

void loop() {
    buttonState = digitalRead(buttonPin);  // read the state of the pushbutton value
    if (buttonState == HIGH) {            // check if the pushbutton is pressed.
        analogWrite(ledPin, 60);           // turn the LED part on if the buttonState is HIGH
    } else {
        analogWrite(ledPin, 255);          // turn LED full on if the buttonState is LOW
    }
}

Upload this sketch to your Arduino and check that it works.

The light should be dimmer when you press button D7.

The analogue outputs can have values from 0 to 255. Try out some different numbers in the analogWrite() function.

What are those numbers, sir?

Digital data is just “on” or “off”, HIGH or LOW, etc.

Analogue data is a number. There’s a minimum and a maximum possible value for that number.

In an Ardiuno, the minimum number is 0 and the maximum is 255.

To complete this activity, change the program so that the LED is on with a value of 50 when the button is not pressed, and is on with a value of 190 when the button is pressed.

Level 2 — The Dimmer Switch

What are we doing in this level, sir?

You’ll learn how to use buttons to make an LED shine brighter and dimmer.

Open a new Arduino window with a blank sketch in it.

Copy the sketch below into your Arduino window. Remember to delete the blank sketch first.

int ledPin = 9;                 // define the pin for LED D9 on the ThinkerShield
int upButtonPin = 7;            // define the pin for the "up" button
int downButtonPin = 6;          // define the pin for the "down" button
int upButtonState = 0;          // create a variable for what the up button is doing
int downButtonState = 0;        // create a variable for what the down button is doing
int brightness = 0;             // create a variable for the brightness of the LED

void setup() {
    pinMode(ledPin, OUTPUT);         // set up the ledPin as an OUTPUT
    pinMode(upButtonPin, INPUT);     // set up the buttonPin as an INPUT
    pinMode(downButtonPin, INPUT);   // set up the buttonPin as an INPUT
}

void loop() {
    upButtonState = digitalRead(upButtonPin);      // read the state of the up pushbutton
    downButtonState = digitalRead(downButtonPin);  // read the state of the down pushbutton
    if (upButtonState == HIGH) {            // check if the up pushbutton is pressed.
        brightness = brightness + /* ADD CODE HERE */;         // increase the brightness by 1
    }
    if (downButtonState == HIGH) {          // check if the down pushbutton is pressed.
        brightness = brightness - /* ADD CODE HERE */;         // decrease the brightness by 1
    }
    if (brightness > 255) {                 // Don't let brightness get bigger than 255
        brightness = 255;
    }
    if (brightness < 0) {                   // Don't let brightness get smaller than 0
        brightness = 0;
    }
    /* ADD CODE HERE */                      // set the brightness of the LED
    delay(10);                               // wait a little while before checking again
}

There are three places that say /* ADD CODE HERE */. Use the comments to work out what to put in these places, instead of /* ADD CODE HERE */.

To set the brightness of the LED, use the analogWrite() function.

Put in analogWrite(ledPin, brightness);
