Level 1 — Change the brightness of an LED

What are we doing in this level, sir?

You are going to learn how to use the analogue output to change the brightness of an LED.

Open a new Arduino window with a blank sketch in it.

Copy the sketch below into your Arduino window. Remember to delete the blank sketch first.

int ledPin = A9                 // define the pin for LED D9 on the ThinkerShield
int buttonPin = 7;              // define the pin for button D7
int buttonState = 0;            // create a variable for what then button is doing

void setup() {
    pinMode(ledPin, OUTPUT);    // set up the ledPin as an OUTPUT
    pinMode(buttonPin, INPut);  // set up the buttonPin as an INPUT
}

void loop() {
    buttonState = digitalRead(buttonPin);  // read the state of the pushbutton value:
    if (buttonState == HIGH) {            // check if the pushbutton is pressed.
        analogWrite(ledPin, 127);          // turn the LED full on if the buttonState is HIGH:
    } else {
        analogWrite(ledPin, 255);          // turn LED half on if the buttonState is LOW:
    }
}

Upload this sketch to your Arduino and check that it works.

Now open the Arduino “serial monitor”. You do this by choosing “Serial monitor” from the “Tools” menu, as shown in the picture.

Adjust the potentiometer (the knob) on the Arduino and look at the data coming in on the serial monitor.

What are those numbers, sir?

The number is how the Arduino knows the position of the potentiometer (or knob). This kind of input is called analogue.

Digital data is just “on” or “off”, HIGH or LOW, etc.

Analogue data is a number. There’s a minimum and a maximum possible value for that number. What are the smallest and biggest numbers you can get?
