Arduino function reference

pinMode(pin, mode);

Tells the Arduino that a pin is an INPUT_PULLUP or an OUTPUT.

delay(time);

Tells the Arduino to wait for the time.

digitalRead(pin);

Finds out if an input is HIGH or LOW.

digitalWrite(pin, value);

Makes an output either HIGH or LOW.

analogRead(pin);

Finds out the value of an analogue input.

analogWrite(pin, value);

Sets an analogue output to the value you choose.

Other code
