Level 1 — Change the blink “on” time

What are we doing in this level, sir?

I am going to give you a sketch that blinks an LED on your Arduino MAAS shield.

Your challenge is to change the sketch so that the LED is on for a shorter time: it’s on for one second now, change it so that it’s on for only half a second.

Start the Arduino IDE.

How do I do that? I think I did it before, but I can’t remember.

You learned how to do this in tutorial C1. Click the link and look at that if you need to.

You might want to arrange your browser and Arduino windows side by side so you can see both of them at the same time.

Copy the sketch below into your Arduino window. Remember to delete the other sketch first.

void setup() {          // the setup function runs once when you press reset or power the board
pinMode(12, OUTPUT);    // initialize digital pin 12 as an output.
}
void loop() {           // the loop function runs over and over again forever
digitalWrite(12, HIGH); // turn the LED on
delay(1000);            // wait for a second
digitalWrite(12, LOW);  // turn the LED off
delay(1000);            // wait for a second
}

See if you can work out how to change the length of time the LED is on. Make a change, upload the sketch to the Arduino, and see if it works right.

Remember, the challenge is to have the LED on for only half a second.

I can give you some hints if you need them, but have a go by yourself first.

Can you see a comment in the sketch that says “wait for a second”? (comments start with //. Everything after // is for people to read, not the computer).

The command delay(1000); tells the Arduino to wait for 1000 milliseconds. 1000 milliseconds is 1 second.

Change the first delay(1000); to delay(500);.

I did it!

That’s fantastic!

Now claim your achievement.

  1. Save your Arduino sketch. Name it C2-level1.
  2. Copy the saved sketch to your Year 8 Digital Tech folder on Google Drive.
  3. Complete a claim slip. The code is C2 and the level is 1.

Level 3 — Blink two LEDs

What are we doing in this level, sir?

I am going to give you a sketch that blinks an LED on your Arduino MAAS shield.

Your challenge is to change the sketch so that another LED (called Pin 8) also blinks on and off.

Open a new Arduino window.

Copy the sketch below into your Arduino window. Remember to delete the other sketch first.

void setup() {          // the setup function runs once when you press reset or power the board
pinMode(12, OUTPUT);    // initialize digital pin 12 as an output.
}
void loop() {           // the loop function runs over and over again forever
digitalWrite(12, HIGH); // turn the LED on
delay(1000);            // wait for a second
digitalWrite(12, LOW);  // turn the LED off
delay(1000);            // wait for a second
}

See if you can work out how to make another LED blink. You will need to make changes in the setup section of the sketch, and in the loop section of the sketch.

I can give you some hints if you need them, but have a go by yourself first.

Wherever you see pin 12 in the sketch, you need to make a second copy of that line and change one of them to use pin 8 instead.

Can you see a line in the sketch that looks like this:

pinMode(12, OUTPUT);    // initialize digital pin 12 as an output.

That tells the Arduino that pin number 12 is an output. You’ll need to add another line just like it for pin number 8.

You also need to look for two lines that start with digitalWrite and make copies that say pin 8 too.

There are two other lines that you need to make copies of and change the 12 to 8. They are

digitalWrite(12, HIGH); // turn the LED on

and

digitalWrite(12, LOW);  // turn the LED off

That’s fantastic!
