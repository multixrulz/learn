Learn Arduino

P1 — Project Options

Utilising your skills in control technologies and coding you are to design, produce and evaluate one of these three options:

Alarm system

Difficulty: Easier

Design Brief

Create an alarm/alert system to inform you if someone has broken into something you own or care for. This may include bedrooms, boxes, treasure chests etc.

Digital dice

Difficulty: Harder

Design Brief

Create a digital dice that allows you to virtually “roll” the dice, and display the result on a set of LEDs arranged in a H pattern like that on a physical dice.

Traffic Lights

Difficulty: Harder

Design Brief

Create a traffic light controller that has red, amber and green lights for two intersecting streets.
